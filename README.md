This repositories is about the Super AI Engineer Season 6 of me for the all advantures of AI Engineer in 2026 career path.

## Level 1
### Hackathon 4: 5 Domains Hackathon
รวม Code และ Notebook สำหรับการแข่งขันโจทย์ 5 โดเมน ได้แก่
- **Heart Disease Prediction**: การใช้ AutoGluon พร้อม Feature Engineering
- **Sleep Staging Classification**: การสกัดฟีเจอร์ Time-Series (BVP, ACC, TEMP) และโมเดล CatBoost
- **Thai Language Image Captioning**: โมเดล Pathumma (Vision-Language) พร้อมเทคนิค Textual Ensemble
- รวมถึงโจทย์อื่นๆ เช่น House Recognition และ Word Segmentation
